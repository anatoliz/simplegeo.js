getLocation(options, callback)
First try get the location from the browser (this will usually ask the user to approve). If that fails or the user denies the request, fall back to doing an IP based lookup for the location
Parameters:
options Optional
W3C Geolocation PositionOptions
callback
See callbacks. The data passed to the callback matches the W3C Geolocation Position Interface.
